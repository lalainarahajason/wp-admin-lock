### F5 — Import / Export de configuration
**Priorité : HAUTE**

**Description**
Exporter l'intégralité de la configuration du plugin en JSON signé. Importer sur un autre site avec validation du format, dry-run (aperçu des changements), et confirmation explicite. Permet de dupliquer une configuration de référence sur plusieurs sites.

**Critères d'acceptation**
- [ ] Export JSON téléchargeable depuis l'interface admin
- [ ] Le fichier JSON est signé avec un hash pour détecter toute altération
- [ ] Import avec validation du schéma JSON avant application
- [ ] Dry-run : affichage des différences config actuelle vs config importée
- [ ] Confirmation explicite requise avant application de l'import
- [ ] Les secrets sensibles (URL admin custom) peuvent être exclus de l'export
