
### F10 — Journal d'activité (Audit Log)
**Priorité : MOYENNE**

## Objectif
Logger les événements de sécurité importants dans une table dédiée (`wp_lebosecu_logs`). Interface de consultation avec filtres, pagination et export CSV. Rétention automatique configurable.

**Critères d'acceptation**
- [ ] Connexions réussies et échouées enregistrées (IP, user-agent, timestamp)
- [ ] Modifications de configuration WP Lebo Secu enregistrées
- [ ] Requêtes bloquées (login, API REST, énumération) enregistrées
- [ ] Interface de consultation avec filtres par type, date, IP
- [ ] Export CSV depuis l'interface admin
- [ ] Rétention configurable (défaut : 30 jours) avec nettoyage automatique
- [ ] Table `wp_lebosecu_logs` créée à l'activation du plugin
