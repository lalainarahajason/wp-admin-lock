### F8 — En-têtes de sécurité HTTP
**Priorité : MOYENNE**

## Objectif 
Injecter automatiquement les headers de sécurité HTTP sur toutes les réponses WordPress. Chaque header est activable/désactivable individuellement et sa valeur est éditable.

**Critères d'acceptation**
- [ ] `X-Frame-Options: SAMEORIGIN` activé par défaut
- [ ] `X-Content-Type-Options: nosniff` activé par défaut
- [ ] `Referrer-Policy: strict-origin-when-cross-origin` activé par défaut
- [ ] `Permissions-Policy` configurable avec valeur par défaut sécurisée
- [ ] `Content-Security-Policy` en mode `report-only` par défaut, strict en option
- [ ] Chaque header peut être désactivé individuellement
- [ ] Valeurs éditables depuis l'interface admin
