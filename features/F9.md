### F9 — Désactivation sélective de fonctionnalités WordPress
**Priorité : MOYENNE**

## Objectif
Toggles indépendants pour désactiver des fonctionnalités WordPress natives constituant des vecteurs d'attaque ou des fuites d'information.

**Critères d'acceptation**
- [ ] Désactivation de l'éditeur de fichiers thèmes/plugins (`DISALLOW_FILE_EDIT`)
- [ ] Désactivation de XML-RPC
- [ ] Désactivation des embeds oEmbed
- [ ] Désactivation des pingbacks/trackbacks
- [ ] Chaque option est un toggle indépendant
- [ ] Avertissement affiché si une désactivation peut impacter un plugin actif
