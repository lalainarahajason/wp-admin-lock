### F3 — Gestionnaire de règles .htaccess
**Priorité : HAUTE**

**Description**
Interface pour créer, éditer, supprimer et ordonner des blocs de règles `.htaccess` personnalisées. Le plugin gère une section balisée (`# BEGIN Lebo Secu` / `# END Lebo Secu`). Backup automatique avant chaque modification. Prévisualisation diff avant confirmation.

**Critères d'acceptation**
- [ ] Les règles sont injectées dans une section balisée et identifiable
- [ ] Backup automatique du `.htaccess` avant chaque écriture
- [ ] Prévisualisation du `.htaccess` final avant application
- [ ] Les règles hors de la section Lebo Secu ne sont jamais modifiées
- [ ] Gestion des erreurs si le `.htaccess` n'est pas accessible en écriture
- [ ] Restauration possible depuis le dernier backup en un clic
